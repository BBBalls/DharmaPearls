Welcome

Dharma Pearls is a repository of freely available modern English translations of Chinese Buddhist texts. While it currently has a limited number of texts available, more will be added every month.

You can start exploring the English translations that are available, check on what's been added to the site since you're last visit, or read about the author's copyright policies and long-term plans.

You can also contact Charles Patton by email with questions, corrections, or suggestions. You can also visit the Dharma Pearls repo on Github and open an issue to report bugs or suggest improvements.
