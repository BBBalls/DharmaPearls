What's New

  • 22-Oct-2019: Published table of contents for The Related Discourses based on Yinshun's reconstructed version. It includes 7 parts and 51 groups of sutras.
  • 14-Oct-2019: Sutras 748-754 and 758-761 have been added to the Related Discourses.
  • 11-Oct-2019: The first "official" update released. Aside from an improved web design, it includes translations of Sutra 29 of the Long Discourses and Sutras 1-7 of the Medium Discourses.
  • 15-Sep-2009: The Dharma Pearls repo on Github goes public and is published on Github Pages. This was a broken web design. The release was simply to walk-through the Github Pages deployment process.
